The Ultimate Local Study Toolkit

StudyForge is a Progressive Web App (PWA) designed to give students powerful, private tools without the bloat of modern frameworks or server processing times. Built entirely with Vanilla JavaScript, HTML5, and CSS3, this application runs 100% locally in your browser.

Core Features:

PDF Engine: Merge multiple PDFs or extract specific pages using pdf-lib.

OCR Scanner: Extract text from photos of handwritten notes locally using Tesseract.js.

Anki Flashcard Maker: Instantly convert text notes into Anki-ready CSV files.

Study Tools: Native Speech Reader, Open Library Citation Builder, and a live Text Analyzer.

Offline Ready: Installable as a PWA with a global Light/Dark Theme Engine.
